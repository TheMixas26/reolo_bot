# Команды

Ниже собран список команд из `dev/varibles/command_list.txt`.

## Основной бот: пользовательские команды

- `/start` — запустить бота
- `/help` — помощь и краткая справка
- `/changelog` — история обновлений
- `/battle` — ссылка на карточного бота
- `/personal_notifications` — включить или выключить личные уведомления о днях рождения
- `/add_birthday` — добавить свой день рождения
- `/me` — профиль пользователя, достижения и баланс
- `/achievements` — список достижений
- `/today` — имперская дата на сегодня
- `/nearest_event` — ближайшие праздники Империи
- `/all_events` — все праздники Имперского календаря

## Основной бот: команды администратора

- `/send_daily` — принудительно отправить общую ежедневку
- `/send_personal_daily` — принудительно отправить личные ежедневки
- `/add_birthday_by_username` — добавить день рождения пользователю
- `/setcmd` — обновить список команд у ботов
- `/edit_currency` — изменить курс валюты
- `/add_achievement` — создать новое достижение
- `/grant_achievement` — выдать достижение пользователю
- `/revoke_achievement` — отозвать достижение у пользователя
- `/add_conditions` — обновить условия достижения
- `/broadcast` — разослать сообщение всем пользователям
- `/fake_post` — опубликовать пост от имени подписчика
- `/send_actual_db` — отправить актуальный бэкап
- `/stop_bot` — остановить бота
- `/send_smth` — тестовая сервисная отправка

## Банковый бот

- `/start` — запустить банкового бота
- `/bank` — открыть меню Имперского банка

## Карточный бот: пользовательские команды

- `/start` — запустить карточного бота
- `/help` — помощь по карточной игре
- `/roll` — купить и открыть пак
- `/packs` — показать доступные паки
- `/wallet` — показать карточный баланс
- `/events` — показать карточные ивенты
- `/inventory` — открыть инвентарь карт
- `/cards` — посмотреть каталог карт
- `/duel` — вызвать игрока на дуэль
- `/team_battle` — начать командный бой
- `/cancel` — отменить текущее действие или бой

## Карточный бот: команды администратора

- `/cg_events_admin` — список всех карточных ивентов
- `/cg_create_event` — создать карточный ивент
- `/cg_close_event` — закрыть карточный ивент
- `/cg_reward_event` — выдать награду участнику ивента
- `/cg_packs_admin` — список паков для администратора
- `/cg_add_pack` — создать новый пак
- `/cg_edit_pack` — изменить параметры пака
- `/cg_add_card` — добавить новую карту
- `/cg_edit_card` — изменить существующую карту
