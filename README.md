# Предложка Империи

Telegram-бот для канала [Имперский Вестник](https://t.me/reolo_channel) и связанных с ним сервисов.

Проект уже давно вышел за рамки обычной предложки. Сейчас в репозитории живут:

- основная система модерации пользовательских постов;
- достижения и пользовательский профиль;
- напоминания о днях рождения;
- погодные уведомления;
- небольшой банк;
- карточная мини-игра и дуэли;
- интеграция с YandexGPT по тегу `#ai`.

## Быстрый старт

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Отредактируйте `dev/settings.py` для названий ботов, валюты и других публичных параметров.

3. Создайте `dev/config.py` со своими токенами и секретами.

4. Запускайте проект из корня репозитория:

```bash
python dev/main.py
```

## Что важно знать

- Основная SQLite-база хранится в `dev/database/bot.sqlite3`.
- Планировщик фоновых задач стартует автоматически при запуске `dev/main.py`.
- В `DEBUG_MODE` основной бот и RPG-бот запускаются всегда, банковый бот только вне debug-режима.
- Несекретные настройки проекта вынесены в `dev/settings.py`.

Если нужен подробный шаблон `config.py` и описание модулей, смотрите [документацию по настройке](doc/setup.md) и [архитектуре](doc/architecture.md).
