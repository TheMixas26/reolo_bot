# Предложения по задачам после обзора кодовой базы

## 4) Улучшение тестов
**Задача:** добавить unit-тесты на валидацию дат в `birthdays.py`.

- Покрыть `add_birthday` кейсами:
  - корректный формат `ДД.ММ`;
  - корректный формат `ДД.ММ.ГГГГ`;
  - некорректная дата (например `31.02`);
  - некорректная строка (например `abc`).
- Проверить, что на ошибочных данных функция возвращает `False`, а на валидных — `True` и сохраняет нормализованные поля `day/month/year`.

**Критерий готовности:** тесты стабильно проходят локально и защищают от регрессий в парсинге дат.
